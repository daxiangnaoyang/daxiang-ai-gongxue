7条场景化路径

你不用从头硬学，先按问题场景选路线。

课程内容覆盖AI通识、提示词、Coze智能体、AI编程、内容生产、职场提效和AIGC创作。每条路径都对应一个清晰的最小成果。

路径 01

刚入门，不知道AI怎么学

先理解大模型、Token、上下文、提示词、Agent的基础关系。

成果：形成第一版AI学习路线。
路径 02

只会问AI，想真正会用AI

从问一个问题，升级为和AI一起完成资料整理、总结、方案或大纲。

成果：写出可复用的任务提示词。
路径 03

想搭建自己的AI Agent

从Agent认知到Coze、工作流、知识库、数据库和更工程化的Agent系统。

成果：做出一个稳定完成单一任务的助手。
路径 04

想用AI做内容和个人IP

建立选题、素材包、大纲、初稿、改稿、分发的内容生产流程。

成果：完成一篇长文或三条短内容。
路径 05

想用AI提升工作效率

从高频重复任务开始，用AI做总结、表格、会议纪要、调研和归档。

成果：做出一个能服务自己工作的AI助手。
路径 06

想学AI编程和Claude Code

理解API、目录、配置、MCP、Skills，让AI带你完成小项目。

成果：能和AI协作改一个项目或工具。
路径 07

想做AIGC短片、绘图和故事

从故事、角色、分镜、画面、视频片段到项目资产管理。

成果：完成一个30-60秒短片或图文故事。

大象AI共学不承诺速成结果，也不制造焦虑。适合愿意实践、愿意复盘、愿意把AI用进真实场景的人。

我是小白，能不能学？

可以。从小白路线开始，不需要一上来学复杂工具。但你需要跟着做，AI是用会的，不是看会的。

我时间不多怎么办？

不用一次性学完。先按自己的场景选一条路线，每天看一小节，然后马上用到工作或内容里。

和其他AI课程有什么区别？

这里更像共学社区，不是买完就结束的录播课。核心是飞书知识库持续更新、社群里边学边问、按你的问题场景给路线。

加入后我具体能得到什么？

持续更新的AI学习知识库、场景化学习路径、社群答疑，以及一套逐步形成自己AI工作流的方法。

哪些人不适合？

只想要资料包、不愿动手、期待立刻获得确定结果、只想学零散技巧的人，不适合这个共学模式。
